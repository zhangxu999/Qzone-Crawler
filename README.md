
Qzone-Crawler
=============

一个支持Chrome内核浏览器的插件,能够抓取并统计QQ空间的说说，评论，赞等。统计分析与空间主人比较亲密的人。

---------------------------------------------

**使用环境**：chrome内核浏览器.(如360，搜狗，猎豹，百度浏览器)。
使用JQuery,Jquery-json，前端插件.
